## The Idea

Instead of "give me 10 post ideas" → take ONE topic and snowball it. One core problem expands into sub-angles, contrarian takes, personal stories, how-tos, myths. **One topic = 30 days of content.**

Depth-first, not breadth-first.

---

## The Math

| Step | What happens | Output |
|------|-------------|--------|
| Prompt 1 | Define one core problem + ICP | 1 seed |
| Prompt 2 | Break into 7 unexpected angles | x7 |
| Prompt 3 | 3 subtopics per angle | x3 = 21 |
| Prompt 4 | 3 formats per subtopic + calendar | x3 = 63 pieces → 30-day calendar |

**1 topic → 7 angles → 21 subtopics → 63+ content pieces**

---

## Prompt 1: The Core

```
You're a content strategist.
Before answering, ask me clarifying questions about:
-- goal
-- target audience
-- pain points
-- constraints
-- tone
-- platforms

Based on my answers, define one core problem in my niche that people
are actually willing to discuss and share.

Deliver:
- why this problem matters (2-3 sharp sentences)
- 30-day content goals (what we measure: saves/leads)
- an ICP table (who / what they want / what blocks them / language / triggers / taboos)

Be specific
```

**Why it works:** Forces Claude to ask YOU questions first instead of guessing. Establishes the single core problem. The ICP table ensures every piece speaks to a real person.

---

## Prompt 2: Angles & Headlines

```
Take the core problem and propose 7 unexpected angles
(counterintuitive, "you're doing it wrong," myth vs reality,
hidden cost of the usual solution).

For each angle provide:
- 2 hooks (12 words or fewer) for Threads/Reels
- 1 core insight (80 words or fewer)
- credibility facts
- 1 potentially controversial statement
```

**Why it works:** One problem becomes 7 distinct angles. The constraints (counterintuitive, myth vs reality, hidden cost) prevent generic "5 tips" content. 12-word hook limit forces punchy, platform-native headlines.

### The 7 Angle Types

| Type | What it is |
|------|-----------|
| **Counterintuitive** | "You're doing it wrong" — flips conventional wisdom |
| **Myth vs Reality** | Debunks common beliefs |
| **Hidden Cost** | Reveals unseen price of the usual solution |
| **Contrarian Take** | Controversial position backed with evidence |
| **Personal Story** | Narrative-driven from real experience |
| **How-To** | Actionable, step-by-step |
| **Taboo/Trigger** | Addresses what the niche avoids talking about |

---

## Prompt 3: Branches into Ready Posts

```
For each of the 7 angles, create 3 subtopics with a
"wow, I never thought of that" effect.

For each subtopic, create 3 formats:
- Educational (hook → insight → actionable steps)
- Provocative (debatable hook → three proofs → question)
- Case study (context → action → numbers → takeaway)
```

---

## Prompt 4: 30-Day Calendar

```
For each format, adapt for Threads / Reels / Telegram
(length, visuals, CTA) and define a success metric for each.

Consolidate everything into a 30-day calendar:
date / channel / topic / format / goal / distribution notes.
Remove any repetition.
```

---

## The 3 Content Formats

| Format | Structure | Best for |
|--------|-----------|----------|
| **Educational** | Hook → Insight → Actionable Steps | Trust, saves |
| **Provocative** | Debatable Hook → 3 Proofs → Question | Engagement, shares, comments |
| **Case Study** | Context → Action → Numbers → Takeaway | Credibility, leads |

---

## Why This Works

1. **Thematic coherence** — content tells a connected story over 30 days
2. **Authority signaling** — covering one topic from 7+ angles shows expertise
3. **Non-repetitive variety** — 3 formats prevent staleness
4. **Platform optimization** — built-in adaptation per channel
5. **Measurability** — each piece has a success metric from the start

## How This Applies to DesignTransform

### Core Problem (Our Seed)

"Small creative shops are drowning in production work and can't get ahead."

### Snowball It

**7 angles on that one problem:**

1. **Counterintuitive:** "Hiring is making your backlog worse, not better"
2. **Myth vs Reality:** "Unlimited revisions doesn't mean unlimited scope"
3. **Hidden Cost:** "The real cost of doing everything yourself isn't money — it's the clients you're losing"
4. **Contrarian:** "You don't need better project management — you need less work on your plate"
5. **Personal Story:** "I ran a shop for 30 years. Here's when I finally stopped drowning."
6. **How-To:** "How to clear a 50-task backlog in 2 weeks without hiring"
7. **Taboo:** "Most small agencies are one bad month from closing and nobody talks about it"

Each of those → 3 subtopics → 3 formats = **63 pieces of content** from ONE problem.

### For the Content Site / SEO Flywheel

This method IS the content engine. Run the snowball on each major pain point. Each produces 30-60 articles. Stack 3-4 pain points and you have 6-12 months of content without running out of ideas.

### As a Skill

This should become a Claude Code skill: `/snowball [topic]` — runs the 4-prompt chain automatically, outputs a 30-day calendar. Sell it. Teach it. Use it.
