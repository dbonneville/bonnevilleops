# DesignTransform — Go-To-Market Strategy

> How we find clients and get them to pay us. Living document.

---

## Market Definition

**Who:** Small creative shops / design shops, 1-10 people, doing production work (WordPress, print, web, email, assets). Drowning in backlog. Not AI-native. Not on Twitter posting about systems.

**Where they are:**
- Local business groups, chambers of commerce
- WordPress meetups, WordCamps
- Facebook groups (Elementor, Divi, WooCommerce, freelancer groups)
- LinkedIn (passive)
- Print trade associations
- Referral networks
- Clutch.co, Sortlist, DesignRush directories

**What they feel:**
- Overwhelmed, behind, can't catch up
- Can't hire (too expensive, too risky, freelancers flake)
- Doing work below their pay grade
- Quality slipping because everyone's stretched thin
- Afraid of losing clients to faster shops

## Channel Strategy

### Channel 1: Cold Email (Primary — Starting Here)

**Why cold email:**
- Direct, measurable, cheap
- Can target exact profile (WordPress shops, print studios, small agencies)
- No algorithm, no content calendar, no posting
- Can validate messaging before scaling

**What we need:**
- [ ] Lead list: scrape directories, LinkedIn, Clutch, Google Maps for small creative shops
- [ ] Email infrastructure: dedicated domain, warmup, sending tool
- [ ] Cold email sequences (3-5 touches)
- [ ] Landing page for replies to visit
- [ ] Tracking: open rates, reply rates, booked calls

**Lead Sources:**
- Clutch.co — filter by size, location, services
- Google Maps — "web design" + city, "graphic design" + city
- LinkedIn Sales Navigator — job title + company size filters
- DesignRush, Sortlist, Agency Spotter directories
- WordPress developer directories
- Local chamber of commerce member lists
- Print trade association directories

**Cold Email Approach:**
- Short, plain text, no HTML
- Lead with their pain, not our features
- No "AI-powered" language
- Sound like a person, not a pitch deck
- Offer something small and concrete

**Sequence Sketch:**

> **Email 1:** Pain hook + offer
> "Running a [design/web] shop with a backlog that never shrinks? I clear overflow work for shops like yours — WordPress fixes, banner batches, email templates, print files. Flat monthly rate, no contracts. Worth a quick look?"

> **Email 2:** (3 days later) Social proof or specificity
> Short follow-up with a concrete example or testimonial

> **Email 3:** (5 days later) Breakup / different angle
> Final touch, different pain point, easy out

### Channel 2: Click Testing (Validation)

Run Steve Pieper click test method to validate which pain hooks resonate before scaling cold email or paid ads.

See: `_planning/steve-pieper-click-test-method-v2.md`

### Channel 3: Content Site / SEO (Medium Term)

- Articles about every pain point creative shops face
- Each article maps to a skill/service we offer
- Automated pipeline: scrape pain → write → publish → rank
- Inbound leads over time

### Channel 4: Referrals & Word of Mouth (Ongoing)

- Every happy client is a referral source
- These shops talk to each other — local networks, meetups, Facebook groups
- Offer referral incentive? [TBD]

### Channel 5: Paid Ads (After Validation)

- Only after click testing validates messaging
- Facebook/Instagram targeting small business owners + design interests
- Google Ads for "WordPress help" / "design overflow" type searches
- Retargeting site visitors

## Cold Email Infrastructure

### What to Set Up

| Component | Tool Options | Notes |
|-----------|-------------|-------|
| **Sending domain** | Separate from main domain | Protect main domain reputation |
| **Email warmup** | Instantly, Warmbox, Lemwarm | 2-3 weeks before sending |
| **Sending tool** | Instantly, Smartlead, Lemlist | Sequences, tracking, A/B testing |
| **Lead scraping** | Apollo, Clay, Phantom Buster, manual | Build targeted lists |
| **CRM** | HubSpot (free), Close, Pipedrive | Track replies and pipeline |
| **Landing page** | Simple page on main site | Where replies go to learn more |

### Metrics to Track

| Metric | Target |
|--------|--------|
| Open rate | 50%+ |
| Reply rate | 5-10% |
| Positive reply rate | 2-5% |
| Booked calls | 1-2% of sends |
| Send volume | 50-100/day after warmup |

## Messaging Framework

### Core Message

**Don't sell AI. Don't sell systems. Sell relief.**

"You have too much work and not enough hands. I clear the overflow. Flat rate. No contracts."

### Pain Hooks to Test (Click Test Round 1 Candidates)

1. "My backlog never actually clears"
2. "I can't find reliable freelancers"
3. "I'm doing work below my pay grade"
4. "I don't have time to learn new tools"
5. "I'm losing clients to shops that move faster"
6. "I stayed up until 2am doing production work"
7. "Every time I finish one task, three more appear"
8. "My team is burned out doing work outside their expertise"
9. "Hiring is a trap — $60K before benefits and they might quit"
10. "I started this business to do creative work, not admin"

### Language Rules

| Do | Don't |
|----|-------|
| Backlog, overflow, pile, behind | Scale, leverage, 10x |
| Relief, breathing room, caught up | Transform, disrupt, innovate |
| Flat rate, no surprises | AI-powered, automation stack |
| Shop, studio, team | Agency, enterprise, ecosystem |
| Doug, person, veteran | System, platform, solution |

## GTM Phases

### Phase 0: Foundation (Now)

- [x] Define target market
- [x] Document pain points
- [ ] Nail the ICP with data
- [ ] Set up cold email infrastructure
- [ ] Write cold email sequences
- [ ] Build simple landing page for cold email traffic
- [ ] Run click tests on pain point hooks

### Phase 1: First Clients (Month 1-2)

- [ ] Send cold emails (50-100/day)
- [ ] Book discovery calls
- [ ] Close 2-3 starter clients at $500-995/mo
- [ ] Deliver great work, collect testimonials
- [ ] Refine intake/delivery workflow based on reality

### Phase 2: Proof & Refinement (Month 3-6)

- [ ] 5-10 active clients
- [ ] Testimonials and case studies
- [ ] Content site publishing articles
- [ ] Raise prices for new clients
- [ ] Skills catalog growing daily
- [ ] Cold email sequences optimized from data

### Phase 3: Growth (Month 6-12)

- [ ] Graduate to $2,500-4,500/mo pricing
- [ ] 10-15 active clients
- [ ] Inbound starting from content/SEO
- [ ] Add CC/OC setup as premium service
- [ ] Referral pipeline active

### Phase 4: Target State (Year 2+)

- [ ] ~14 clients at $7K/mo = $100K/mo
- [ ] Known expert in AI for creative shops
- [ ] Skills catalog is THE resource for this niche
- [ ] Content site drives steady inbound
- [ ] Selective about clients — can choose who to work with
